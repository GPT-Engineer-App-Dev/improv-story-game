# improv-story-game

I want to make an improv game website. It's a game between the the user and a website. The user and the website will take turns appending a word to a string to create an improvised story together. For now the website should sample a word randomly out of a dictionary.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App-Dev/improv-story-game.git
cd improv-story-game
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
